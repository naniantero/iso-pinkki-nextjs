# Iso Pinkki site project powered by Next.js

### Relies on
* SSR
* React Query (state mgmt)
* Theme UI (styling)
* Luxon (dates)
* Next-Intl (i18n)
  
  
## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:8008](http://localhost:8008) with your browser to see the result.
